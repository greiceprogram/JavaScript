Introdução ao JavaScript↓ criado em 1995.


JavaScript é uma linguagem interpretada, (rodada em tempo real, código disponibilizado instantaneamente).Baseada em protótipos são bases de estruturas de dados do JavaScript, um conjunto de funções em comum, Multiparadigma.

Comumente utilizada em aplicações web client-side→ parte da aplicação que tem interação com o usuário.

Segue o padrão ECMAScript: Um padrão para varias linguagens.
